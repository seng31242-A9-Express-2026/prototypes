# A9 Express — UI Prototype

A9 Express is a UI/UX prototype designed to deliver a fast, intuitive experience for the A9 Express platform. This repository contains the design prototype built in Figma, covering the complete user interface flow.

## 📌 Overview

This repository documents the UI prototype for A9 Express. The prototype was created using Figma and demonstrates the complete user journey across all key screens of the application.

---

## 🖥️ Screens Covered

- Onboarding / Splash Screen
- Login & Registration
- Home / Dashboard
- Tracking / Status Screen
- Order / Booking Flow
- Profile & Settings
- Notifications

> Note: Screens may vary based on the final design scope.

## 🚀 How to View the Prototype

1. Click the **[Interactive Prototype](https://www.figma.com/proto/eoTviiGxYCNDiYYuFRMIjZ/A9-Express?node-id=52-1094&p=f&viewport=1101%2C-605%2C0.11&t=F72QCH38PLczkEXV-1&scaling=min-zoom&content-scaling=fixed&starting-point-node-id=52%3A1094&show-proto-sidebar=1&page-id=0%3A1)** link above.
2. The prototype will open in your browser — no Figma account required.
3. Click through the screens using the highlighted interactive areas.
4. Use the sidebar to navigate between different flows.
